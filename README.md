# TaskEscrow

TaskEscrow is a modern, blockchain-powered escrow platform for freelance work. It ensures that freelancers get paid for their work and clients only pay when they are satisfied.

## 🚀 How to Run the Web App

We've set up a beautifully designed, modern frontend for the application.

1. **Navigate to the frontend folder**
   Open your terminal and run:
   ```bash
   cd SafeGig/frontend
   ```

2. **Configure your Environment**
   Open the `.env.local` file inside the `frontend` folder and paste your Project ID next to `NEXT_PUBLIC_PROJECT_ID`.

3. **Start the Development Server**
   Run the following command:
   ```bash
   npm run dev
   ```

4. **View the App**
   Open your browser and navigate to [http://localhost:3000](http://localhost:3000) to see the new UI!
